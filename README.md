# TicTacToe_2017_03_Codecool
TIC-TAC-TOE advanced

Optimalizálni az alap programot: egyszerűsítések és átláthatósági javítások eszközölése.

Új Feature-ök:
Computer ellenfél
Választási opciók a játék elején: másik játékos ellen, computer ellen
Könnyű computer ellenfél: teljesen random választ szabad mezőt magának amíg a játék tart
Pont számláló
Opcionális fejlesztés:
Nehéz ellenfél: figyeli a pályát és kiválaszt egy optimális lépést az aktuális nyerési kombinációkból
